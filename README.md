## Features
- 사용자
  - Email, PW, Nickname 회원가입
  - 일반 로그인
  - 회원정보 수정 (미완성)

- 게시판
  - 게시글 및 댓글 기본 CRUD
  - 키워드, 작성자 검색
  - 페이징
  - 좋아요 (미완성)

## Tech Stack
- Java 11 / Spring Boot v2.6.7
- Maven
- Postgresql (DB)

## Dependencies

- Spring Web
- Spring Data JPA
- Validation
- Spring Boot DevTools
- Spring Security
- Thymeleaf
- Lombok
- Postgresql Driver
