## Features
- 사용자
  - Email, PW, Nickname 회원가입
  - 일반 로그인
  - 회원정보 수정 (미완성)

- 게시판
  - 게시글 및 댓글 기본 CRUD
  - 키워드, 작성자 검색

- Postgresql (DB)

- Spring Boot DevTools
- Spring Security
